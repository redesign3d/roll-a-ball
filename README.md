# Roll a Ball (Unity)

A Unity learning project for the classic “Roll a Ball” game.

## Getting the project

Clone the repo and pull updates:

```bash
git clone https://github.com/redesign3d/roll-a-ball.git
cd roll-a-ball
git pull
```

## Setup on a new machine

1. Install Unity Hub.
2. Install Unity **6000.3.11f1** (the version used by this project).
3. In Unity Hub, click **Open** and select the project folder.
4. Let Unity import assets and generate the Library the first time you open it.

## Contributing

1. Fork the repository.
2. Create a feature branch: `git checkout -b your-branch-name`.
3. Make changes and commit with clear messages.
4. Push your branch and open a Pull Request.

Please avoid committing generated Unity folders like `Library/`, `Temp/`, or `obj/`.
